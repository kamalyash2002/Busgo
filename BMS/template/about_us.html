Welcome To Busgo.in


India's Leading Bus Service Provider...



This website was created jointly by Yash Saxena and Shashwat Ajay of class XII-A of G.R.M. School, Bareilly as a part of computer project given by Anil Sir as per the guidelines issued by Central Board of Secondary Education.

This project of Bus Management System website entitled "Busgo.in " after its creators is made on Python based website-framework 'Django' with MySQL as background database.


Our Features



Manage Routes & Schedules



One can add all bus route start and end points (cities, locations, etc.) one's company serves. Manage bus schedules. Set recurring time plans for each day of the week.



Set Ticket Types & Prices



From each bus profile one can add different ticket types - children, adults, pensioners, students, etc., and set standard and custom ticket prices (round-trip, special discounts).



Add Bus Types & Seat Maps



One can create multiple bus profiles containing interactive seat maps so that your customers can easily pick their preferred seats in advance.



Manage Bus Bookings



One can review all bus ticket reservations in one place. See customer details. Set automatic status depending on payment. Add reservations manually.



Customize Bus Tickets



One can create a personalized bus ticket layout by adding it's company brand elements, and a designated place for trip, customer, and booking details using a handy HTML and CSS editor.



Take Payments Online



One can have a wide choice of built-in payment gateways as well as the classic wire transfer and cash payments. Make it convenient for your customers!
